Detailed Description

Represents a module for handling EAI Messages / SMOM Server.

This class is designed to manage EAI-related functionality, such as connecting to a WebSocket server, receiving messages, and handling system alarms. It extends TCMComponentClass to integrate with the broader system framework. Typical usage involves initializing the module, establishing a connection to the server, and processing incoming messages. The class also registers specific message IDs for handling system-level events.
